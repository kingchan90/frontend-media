# MessageMedia frontend coding challenge 

## Installation
Use the package manager [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/) to install project.

```
npm i
```
or 
```
yarn
```
## Start project

```
npm run start
```
or 
```
yarn start
```
